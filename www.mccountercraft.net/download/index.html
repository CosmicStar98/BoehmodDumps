Minecraft Launcher

The vanilla version of CounterCraft can be run on the official Minecraft Launcher as long as you have a valid version of Forge 1.6.4 installed.


Technic Launcher

The Technic Launcher is a faster alternative to running mods, as it installs the mods and runs the forge instance for you. Just simply download the Technic Launcher here, then search for "CounterCraft", or click the button below to head straight to the modpack page.


Twitch Launcher

Thw Twitch Launcher is a relatively new way of playing Mods. Like the Technic Launcher, it will install forge and the CounterCraft mod for you without having to install any external tools. You can get the Twitch Client here, or view the modpack using the button below.
